## Sobre o Projeto
Este repositório foi criado por mim como parte de um desafio de Python aplicado aos alunos, cujo objetivo era desenvolver uma pequena aplicação gráfica utilizando **Tkinter** e publicar o código no GitHub. A proposta introduz os fundamentos de interfaces GUI, estruturação de projetos e boas práticas de versionamento.

## Objetivo Pedagógico
A atividade foi desenvolvida para consolidar o entendimento dos alunos sobre:
- Criação de janelas gráficas com Tkinter  
- Manipulação de widgets básicos  
- Execução de aplicações Python interativas  
- Publicação de projetos em repositórios GitHub  

Além do aprendizado prático, este exercício reforça autonomia, atenção ao detalhe e compreensão de fluxos reais de desenvolvimento.

## Importância Técnica
Tkinter é um dos módulos mais acessíveis e eficientes para introduzir conceitos de GUI em Python. Ao completar este desafio, os alunos puderam:
- Entender a estrutura básica de um loop gráfico  
- Trabalhar com elementos visuais e organização de layout  
- Desenvolver aplicações responsivas e funcionais  

Este projeto demonstra domínio de fundamentos essenciais para ambientes acadêmicos e profissionais.

## Estrutura
O repositório contém um arquivo principal com a construção da janela em Tkinter, mantendo o código simples, organizado e didático.
